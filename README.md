## ğŸ“¦ Build and Deployment

This project is configured for **Static Export** to deploy on GitHub Pages.

### Build Locally
```bash
npm run build
```
This will generate an `out` directory with the static files.

### Deploy to GitHub Pages
This repository includes a GitHub Actions workflow (`.github/workflows/deploy.yml`) that automatically builds and deploys the site when you push to the `main` branch.

1.  Push your changes to GitHub.
2.  Go to **Settings > Pages** in your repository.
3.  Ensure **Source** is set to **GitHub Actions**.
